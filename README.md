> O guardião das suas variáveis de ambiente — escaneia, valida e gera `.env` automaticamente!

---

## ✨ Funcionalidades

- 🔍 **Escaneia** todo o projeto em busca de `process.env.VARIAVEL`
- 📋 **Compara** com `.env` e `.env.example`
- ⚠️ **Mostra** o que está faltando no `.env`
- ✅ **Indica** o que já está configurado
- 📦 **Gera** `.env` automaticamente com todas as variáveis necessárias
- 📊 **Sidebar** com status visual de cada variável
- 🔄 **Atualização** automática ao salvar arquivos

---

## 📦 Instalação

### Via Marketplace (em breve)
1. Abrir VS Code
2. Ir em Extensions (`Ctrl+Shift+X`)
3. Buscar por `Albertool Sentinel`
4. Clicar em Install
